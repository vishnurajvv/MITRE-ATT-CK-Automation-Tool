# MITRE ATT&CK Automation Tool

An AI-powered cybersecurity tool that automatically extracts
adversary Tactics, Techniques, and Procedures (TTPs) from
threat intelligence reports and maps them to MITRE ATT&CK
techniques without human expert involvement.

## Features
- Automatic TTP sentence extraction using NLP
- Machine learning-based MITRE ATT&CK classification
- FastAPI backend for automation
- Structured, SOC-ready output
